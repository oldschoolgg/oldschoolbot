You can gain quest points by doing quests. To get the quest cape, you need to get the maximum QP you can from all available quests.

## Commands

- [[/quest]]: start a random quest you have the requirements for
- [[/quest name\: cook's assistant]]: start a specific quest
- [[/quest name\: cook's assistant info\: true]]: get information for a named quest
- [[/quest claim\:true]]: automatically mark quests as completed (in chronological order) up to your legacy quest points

## Boosts

- 10% speed boost for equipping full Graceful **(can be equipped in any setup)**
