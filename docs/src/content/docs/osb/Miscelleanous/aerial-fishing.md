# Aerial Fishing

You can start aerial fishing with either `+aerial` or `+aerialfish`

You can purchase buyables via `+aerial buy name_of_item`

**Aerial Fishing Buyables**

| :-------------------- | :------------------: |
| Angler hat            |         100          |
| Angler top            |         100          |

| Angler boots          |         100          |
| Pearl fishing rod     |         100          |
