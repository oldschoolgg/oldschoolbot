Aerial Fishing is a combined [[fishing]] + [[hunter]] training method.

Requirements:
- [[hunter:35]]
- [[fishing:43]]

Start with:
[[/activities aerial_fishing]]

You can buy rewards with [[/buy]] and sell [[Golden tench]] for 100 [[Molch pearl]] each using [[/sell items\:Golden tench]].
