## /raid cox start

Start a Chambers of Xeric trip

**Usage**

- `/raid cox start type:<string> challenge_mode:<boolean>? max_team_size:<integer>? quantity:<integer>?`

**Parameters**

| Name | Type | Required | Description | Choices |
| --- | --- | --- | --- | --- |
| `type` | `string` | Yes | Choose whether you want to solo, mass, or fake mass. | solo, mass, fakemass |
| `challenge_mode` | `boolean` | No | Choose whether you want to do Challenge Mode. | - |
| `max_team_size` | `integer` | No | Choose a max size for your team. | - |
| `quantity` | `integer` | No | The quantity to do. | - |
