Sell items from your bank to the bot for GP.

**Usage**

- `/sell items:<string>? filter:<string>? search:<string>?`

**Parameters**

| Name | Type | Required | Description | Choices |
| --- | --- | --- | --- | --- |
| `items` | `string` | No | The items you want to sell (e.g. 1 trout, 5 coal | - |
| `filter` | `string` | No | The filter you want to use. | - |
| `search` | `string` | No | A search query for items in your bank to sell. | - |

**Examples**

- `/sell items:10k trout, 5 coal`
