Sacrifice items from your bank to the bot.

**Usage**

- `/sacrifice items:<string>? filter:<string>? search:<string>?`

**Parameters**

| Name | Type | Required | Description | Choices |
| --- | --- | --- | --- | --- |
| `items` | `string` | No | The items you want to sacrifice. | - |
| `filter` | `string` | No | The filter you want to use. | - |
| `search` | `string` | No | A search query for items in your bank to sacrifice. | - |

**Examples**

- `/sacrifice items:10k trout, 5 coal`
