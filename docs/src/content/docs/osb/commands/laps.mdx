Do laps on Agility courses to train Agility.

**Usage**

- `/laps name:<string> quantity:<integer>? alch:<boolean>?`

**Parameters**

| Name | Type | Required | Description | Choices |
| --- | --- | --- | --- | --- |
| `name` | `string` | Yes | The course you want to do laps on. | - |
| `quantity` | `integer` | No | The quantity of laps you want to do (optional). | - |
| `alch` | `boolean` | No | Do you want to alch while doing agility? (optional). | - |

**Examples**

- `/laps name:Ardougne rooftop course`
