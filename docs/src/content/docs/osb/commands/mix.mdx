Mix potions to train Herblore.

**Usage**

- `/mix name:<string> quantity:<integer>? wesley:<boolean>? zahur:<boolean>?`

**Parameters**

| Name | Type | Required | Description | Choices |
| --- | --- | --- | --- | --- |
| `name` | `string` | Yes | The potion you want to mix. | - |
| `quantity` | `integer` | No | The quantity you want to mix (optional). | - |
| `wesley` | `boolean` | No | If available, pay Wesley to crush items. (optional). | - |
| `zahur` | `boolean` | No | If available, pay Zahur to clean herbs. (optional). | - |

**Examples**

- `/mix name:Prayer potion`
