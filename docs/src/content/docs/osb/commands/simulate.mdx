## /simulate cox

Simulate Chambers of Xeric.

**Usage**

- `/simulate cox quantity:<integer> points:<integer>? team_size:<integer>? challenge_mode:<boolean>?`

**Parameters**

| Name | Type | Required | Description | Choices |
| --- | --- | --- | --- | --- |
| `quantity` | `integer` | Yes | The amount of raids to simulate. | - |
| `points` | `integer` | No | How many points to have (default 25k). | - |
| `team_size` | `integer` | No | The size of your team (default 4). | - |
| `challenge_mode` | `boolean` | No | Challenge mode raids? (default false) | - |
