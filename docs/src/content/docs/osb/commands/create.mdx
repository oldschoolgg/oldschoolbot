Allows you to create items, like godswords or spirit shields - and pack barrows armor sets.

**Usage**

- `/create item:<string> quantity:<integer>? showall:<boolean>?`

**Parameters**

| Name | Type | Required | Description | Choices |
| --- | --- | --- | --- | --- |
| `item` | `string` | Yes | The item you want to create/revert. | - |
| `quantity` | `integer` | No | The amount you want to create. | - |
| `showall` | `boolean` | No | Show all creatable items. | - |
