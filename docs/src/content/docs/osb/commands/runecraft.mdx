Sends your minion to craft runes with essence, or craft tiaras.

**Usage**

- `/runecraft rune:<string> quantity:<integer>? usestams:<boolean>? daeyalt_essence:<boolean>? extracts:<boolean>?`

**Parameters**

| Name | Type | Required | Description | Choices |
| --- | --- | --- | --- | --- |
| `rune` | `string` | Yes | The Rune/Tiara you want to craft. | - |
| `quantity` | `integer` | No | The amount of runes/tiaras you want to craft. | - |
| `usestams` | `boolean` | No | Set this to false to not use stamina potions (default true) | - |
| `daeyalt_essence` | `boolean` | No | Set this to true to use daeyalt essence (default false) | - |
| `extracts` | `boolean` | No | Set this to true to use extracts (default false) | - |

**Examples**

- `/runecraft`
