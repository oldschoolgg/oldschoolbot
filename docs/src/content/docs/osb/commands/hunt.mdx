Hunt creatures with the Hunter skill.

**Usage**

- `/hunt name:<string> quantity:<integer>? hunter_potion:<boolean>? stamina_potions:<boolean>?`

**Parameters**

| Name | Type | Required | Description | Choices |
| --- | --- | --- | --- | --- |
| `name` | `string` | Yes | The creature you want to hunt. | - |
| `quantity` | `integer` | No | The quantity you want to hunt (optional). | - |
| `hunter_potion` | `boolean` | No | Do you want to use Hunter potions for this trip? | - |
| `stamina_potions` | `boolean` | No | Use stamina potions for Herbiboar? | - |

**Examples**

- `/hunt name:Ferret`
