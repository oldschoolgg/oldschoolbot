Giveaway items from your bank to other players.

**Subcommands**

- `/giveaway start`
- `/giveaway list`

## /giveaway start

Start a giveaway.

**Usage**

- `/giveaway start duration:<string> items:<string>? filter:<string>? search:<string>?`

**Parameters**

| Name | Type | Required | Description | Choices |
| --- | --- | --- | --- | --- |
| `duration` | `string` | Yes | The duration of the giveaway (e.g. 1h, 1d). | - |
| `items` | `string` | No | The items you want to giveaway. | - |
| `filter` | `string` | No | The filter you want to use. | - |
| `search` | `string` | No | A search query for items in your bank to giveaway. | - |

## /giveaway list

List giveaways active in this server.

**Usage**

- `/giveaway list`

**Parameters**

_No parameters._

**Examples**

- `/giveaway items:10 trout, 5 coal time:1h`
