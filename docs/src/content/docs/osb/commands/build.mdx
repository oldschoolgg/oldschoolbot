Sends your minion to train Construction by building things.

**Usage**

- `/build name:<string> quantity:<integer>?`

**Parameters**

| Name | Type | Required | Description | Choices |
| --- | --- | --- | --- | --- |
| `name` | `string` | Yes | The object you want to build. | - |
| `quantity` | `integer` | No | The quantity you want to build (defaults to max). | - |

**Examples**

- `/build name:Crude chair`
