Check the stats of a OSRS account.

**Usage**

- `/stats username:<string> type:<string>? virtual:<boolean>?`

**Parameters**

| Name | Type | Required | Description | Choices |
| --- | --- | --- | --- | --- |
| `username` | `string` | Yes | The RuneScape username of the account. | - |
| `type` | `string` | No | The type of account (normal, ironman, HCIM, DMM, leagues, etc). | - |
| `virtual` | `boolean` | No | Show virtual stats? (Up to level 120) | - |
